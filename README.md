# 未来裂隙（Future Rift）

一个基于 **Three.js** 打造的未来感 3D 肉鸽生存射击游戏原型。玩家将在霓虹废墟战场中迎战不断升级的怪物潮，通过 **波次推进、任务奖励、商店购买、随机升级、武器解锁** 持续强化自身，并在战斗过程中逐步回收剧情档案，拼凑出“未来怪物入侵”的真相。

注意，带v的数字版本最大，代表游戏越新，体验也越好，目前最新的是v3

---

## 项目简介

**未来裂隙** 是一个融合了以下元素的网页游戏 Demo：

- **第一人称 / 3D 射击**：使用鼠标控制视角，实时战斗
- **Roguelike 成长循环**：每次升级提供随机强化选项，形成不同构筑
- **波次生存**：敌人随波次增强，威胁等级持续提升
- **任务系统**：通过击杀、存活、推进波次等方式获得额外奖励
- **商店系统**：可购买回血、护盾、增伤、射速和武器解锁等成长项
- **剧情档案收集**：怪物有概率掉落档案卡片，推动主线叙事

该项目适合作为以下方向的展示作品：

- Three.js / WebGL 游戏原型
- 浏览器端动作游戏玩法验证
- 肉鸽成长系统与战斗循环设计
- 前端交互 + 视觉风格化 UI 实践

## 核心特色

### 1. 肉鸽式成长循环
玩家可通过击杀敌人获得积分与经验值，升级时会弹出随机强化选项，例如：

- 提高伤害与暴击率
- 提高射速与移动速度
- 增加最大生命与护盾
- 获得持续回复能力
- 提升暴击伤害或弹体判定

每一局都能形成不同的战斗构筑。

### 2. 波次推进与威胁升级
游戏采用波次生存结构，随着波次推进：

- 敌人数量增加
- 敌人强度提升
- 威胁等级上升
- 战斗节奏持续加快

玩家需要在资源管理与输出构筑之间做出取舍。

### 3. 商店系统
在战斗间隙可进入战术商店，用积分购买本局成长内容，包括：

- 纳米修复（回血）
- 强化外骨骼（提升最大生命）
- 偏移护盾（提升护盾上限）
- 武器过载（增伤）
- 伺服调优（加快射速）
- 纳米注射器（增加主动回血次数）
- 解锁电弧连发 / 磁轨长枪

### 4. 主线叙事嵌入战斗流程
本作不仅是纯生存射击，还加入了“档案回收”叙事机制：

- 怪物死亡后有概率掉落剧情卡片
- 玩家可逐步解锁档案内容
- 故事围绕“维度裂隙”“未来生命体”“时间线改写”展开

这让战斗推进与剧情探索形成统一闭环。

### 5. 未来霓虹风 UI 与战场表现
项目采用高对比霓虹视觉风格，包含：

- HUD 血量 / 护盾 / 等级 / 积分显示
- 武器状态面板
- 商店与升级卡片式弹窗
- 波次提示 Banner
- 剧情档案展示界面
- Bloom 光效与科技感场景氛围

## 武器系统

当前版本包含 4 把风格差异明显的武器：

| 编号 | 武器名 | 特点 |
| --- | --- | --- |
| 1 | PULSE RIFLE（脉冲步枪） | 默认武器，均衡稳定 |
| 2 | SCATTER GUN（散射霰炮） | 近距离爆发，高扩散 |
| 3 | ARC BLASTER（电弧连发） | 高频输出，适合清怪 |
| 4 | RAIL LANCE（磁轨长枪） | 低射速高伤害，适合点杀 |

不同武器与随机升级、商店强化之间可以组合出不同玩法方向。

---

## 敌人与战斗设计

当前版本包含多种敌人原型：

- **浮游刃虫（drone）**：高速近战单位
- **裂甲重锤（brute）**：高血量近战重装单位
- **离子喷吐者（spitter）**：远程攻击单位
- **相位猎影（phantom）**：高机动突进单位
- **棱镜巨兽（titan）**：高威胁大型单位

同时还设计了不同敌人层级：

- 常规体
- 畸变体
- 精英体
- 裂隙主宰

这些层级会影响敌人的生命、伤害、速度与奖励，使后期战斗更有压迫感。

## 操作方式

| 按键 | 功能 |
| --- | --- |
| `W A S D` | 移动 |
| `鼠标` | 视角控制 / 射击 |
| `鼠标左键` | 开火 |
| `1 / 2 / 3 / 4` | 切换武器 |
| `R` | 使用纳米回血 |
| `B` | 打开 / 关闭商店 |
| `Tab` | 暂停 / 解锁鼠标 |
| `Esc` | 关闭界面或进入暂停状态 |

## 运行方式

### 方式一：直接浏览器打开
将 HTML 文件放入项目根目录后，直接用现代浏览器打开。

### 方式二：本地静态服务器（推荐）
推荐使用本地服务器启动，以获得更稳定的模块加载体验。

例如：

```bash
python -m http.server 8000
```

然后访问：

```text
http://localhost:8000
```
